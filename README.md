# paperclip-smusher

JPEG and PNG compression processor for Paperclip.

## Requirements

paperclip-smusher requires Paperclip gem, [jpegtran](http://jpegclub.org) and [optipng](http://optipng.sourceforge.net/) libraries.

## Installation

Add to your Gemfile

````ruby
gem "paperclip-smusher", "~> 0.1.1"
````
